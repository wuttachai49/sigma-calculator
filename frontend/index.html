Sigma Metric Guide

Formula

σ = (TEa% − |Bias%|) / CV%

Westgard QC Rules Reference

RuleTriggerPurpose
1:2s1 control > ±2 SDWarning rule
1:2.5s1 control > ±2.5 SDWarning (tighter)
1:3s1 control > ±3 SDReject — random error
2:2s2 consecutive > +2 SD or both < −2 SDReject — systematic error
R:4sRange between 2 controls > 4 SDReject — random error
4:1s4 consecutive > ±1 SD (same side)Reject — systematic drift
8x / 10x8 or 10 consecutive on same side of meanReject — sustained bias

Quality Goal Index (QGI)

QGI = (Bias% / 1.65) / (CV% / TEa% × 0.5)
QGIInterpretationPriority Action
< 0.8Precision-limitedReduce CV — check reagent lots, pipetting, instrument precision
0.8 – 1.2MixedBoth bias & imprecision contribute — review calibration and precision
> 1.2Accuracy-limitedReduce Bias — investigate calibration, matrix effects, EQA performance
